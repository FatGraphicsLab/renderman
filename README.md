# RenderMan Testbed


## Using Aqsis

### Install Aqsis 1.82

* [github][1]
* [sourceforge download][2] 

### Usage

```
> cd 00-helloworld
> aqsl white.sl
> aqsis white.rib
```


## Using Pixie

### Install Pixie 2.2.6

* [sourceforge][3]
* [sourceforge download][4]

## Usage

```
> start_env.cmd

> cd 00-helloworld
> sdrc white.sl
> rndr white.rib
```
