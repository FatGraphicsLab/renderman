# RenderMan Testbed


## Using Aqsis

### Install Aqsis 1.82

* [github][1]
* [sourceforge download][2] 

### Usage

```
> cd 00-helloworld
> aqsl helloworld.sl
> aqsis sphere.rib
```


## Using Pixie

### Install Pixie 2.2.6

* [sourceforge][3]
* [sourceforge download][4]

## Usage

```
> cd 00-helloworld
> sdrc helloworld.sl
> rndr sphere.rib
```
